# 🍡 冬至汤圆团圆小活动

一个精美的冬至汤圆活动宣传网页，采用现代化设计风格，展示活动详情、制作材料、摆盘艺术等内容。

![冬至汤圆活动网页](https://img.shields.io/badge/Status-Active-success?style=for-the-badge)
![HTML5](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS3](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)

## 🌟 特色功能

- ✨ **精美设计**：温馨的红橙色系配色，搭配淡绿色图标
- 📱 **响应式布局**：完美支持桌面端和移动端设备
- 🎨 **流畅动画**：卡片渐入、悬停效果等丰富的交互动画
- 🎯 **清晰架构**：活动介绍、时间、构思、材料、摆盘、目标一目了然
- 💫 **毛玻璃效果**：30% 透明度卡片配合背景模糊效果

## 📅 活动信息

- **日期**：2025年12月21日（星期六）
- **时间**：晚上 9:00
- **形式**：线上活动
- **内容**：参与人员上线介绍自己的构思及作品

## 📋 页面内容

### 1. 活动介绍
介绍冬至文化背景和活动目的，展示活动亮点。

### 2. 活动时间
明确标注活动日期、时间和内容安排。

### 3. 活动构思
详细说明活动的核心理念、目标对象和特色。

### 4. 所需材料
- 主要材料（糯米粉、温水等）
- 内馅材料（芝麻馅、花生馅等）
- 甜汤底材料
- 咸汤底材料

### 5. 摆盘艺术
- 甜汤摆盘技巧
- 咸汤摆盘技巧
- 创意摆盘方法
- 拍照小技巧

### 6. 活动目标与期许
文化传承、情感连结、技艺学习、创意发挥、美好回忆。

## 🛠️ 技术栈

- **HTML5**：语义化标签结构
- **CSS3**：
  - Flexbox / Grid 布局
  - CSS 变量管理主题色
  - CSS 动画和过渡效果
  - backdrop-filter 毛玻璃效果
- **Google Fonts**：Noto Sans SC / Noto Serif SC

## 🚀 使用方法

### 在线访问

直接访问 GitHub Pages 托管的网页：
```
https://你的用户名.github.io/winter-solstice-tangyuan/
```

### 本地运行

1. 克隆仓库：
```bash
git clone https://github.com/你的用户名/winter-solstice-tangyuan.git
```

2. 进入项目目录：
```bash
cd winter-solstice-tangyuan
```

3. 用浏览器打开 `index.html` 文件即可查看

## 🎨 设计特点

### 色彩方案
- **主色调**：红橙色系（#D94A3D, #F47C3C）
- **辅助色**：淡绿色图标背景（#A8E6CF, #7FD8BE）
- **背景色**：奶油色到淡粉色渐变
- **强调色**：金色（#E8A959）

### 排版
- **标题字体**：Noto Serif SC（优雅衬线体）
- **正文字体**：Noto Sans SC（清晰无衬线体）
- **响应式设计**：适配各种屏幕尺寸

### 视觉效果
- 卡片渐入动画
- 悬停交互效果
- 文字阴影增强立体感
- 背景装饰圆圈动画
- 30% 透明度毛玻璃卡片
